# 💰 Expense Manager

A modern Flutter application that helps users manage their daily expenses efficiently. The app allows users to organize expenses by category, visualize spending patterns through charts, and maintain complete control over their financial records.

## ✨ Features

* 🔐 Secure User Authentication
* 📂 Create, Update, and Delete Expense Categories
* 💵 Add, Edit, Delete, and Restore Transactions
* 🗑️ Trash Management with Restore Functionality
* 📊 Interactive Pie Chart for Expense Analysis
* 📅 Monthly Expense Tracking
* 🔍 Category-wise Expense Summary
* 🎨 Modern Material UI
* ⚡ Fast and Responsive Performance
* 🌐 REST API Integration
* 🔄 Real-time UI Updates using Provider State Management

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/expense-manager.git
```

2. Navigate to the project

```bash
cd expense-manager
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
